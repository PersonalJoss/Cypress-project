# Cypress-project Jocselyn Zumbado Chartier.

This repository contains automated tests for a personal project on the Amazon website. This code is in Cypress/js and it has all the necessary steps to be executed.
The code is commented to make it easier to understand the tests.

## Steps
1. Navigate to “https://www.amazon.com/”
2. Search by name using the filter
3. Search by category using the menu
4. Change the language of the page

### Prerequisites
To use npm, you need to install Node.js v24 and higher


## Installation
Clone this repo and navegate into it.
* git clone https://github.com/JossZC/Cypress-project.git
* cd Cypress-project

You need to run the following command to install dependencies.
* npm install

## Usage
Open the Cypress panel with:
* npx cypress open

or you can run just the exact file:
* npm run test:e2e

Importante note: 
* If it is you first npm execution, maybe you will need to open the Cypress panel it twice.
